<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold dark:text-white text-gray-900 mb-4">Quick start</h2>
  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">Get Friday Night Funkin' running on your machine in minutes</p>
  
  <Steps>
    <Step title="Install dependencies">
      Download and install Haxe from [Haxe.org](https://haxe.org) and Git from [git-scm.com](https://www.git-scm.com). Make sure you have the required build tools for your platform.
    </Step>
    
    <Step title="Clone the repository">
      Clone the repository and initialize submodules to download game assets:
      
      ```bash
      git clone https://github.com/FunkinCrew/funkin.git
      cd funkin
      git submodule update --init --recursive
      ```
    </Step>
    
    <Step title="Install Haxe libraries">
      Install hmm (Haxe Module Manager) and all required libraries:
      
      ```bash
      haxelib --global install hmm
      haxelib --global run hmm setup
      hmm install
      haxelib run lime setup
      ```
    </Step>
    
    <Step title="Build and run">
      Compile and launch the game for your platform:
      
      ```bash
      lime test windows
      # or: lime test mac, lime test linux, lime test html5
      ```
      
      <Note>
        On first build, you may need to run `lime rebuild <platform>` and `lime rebuild <platform> -debug` to compile native dependencies.
      </Note>
    </Step>
  </Steps>
</div>
